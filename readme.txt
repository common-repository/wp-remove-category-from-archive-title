== Description ==
WP Remove Category from Archive Title helps you hide and remove the in-built Category: keyword from the archive.php and categories. 

== Installation ==
* Download wp-remove-category-from-archive-title.zip from https://downloads.wordpress.org/plugin/wp-remove-category-from-archive-title.zip.
* Upload "wp-remove-category-from-archive-title" to the "/wp-content/plugins/" directory.
* Activate the plugin through the "Plugins" menu in WordPress.
